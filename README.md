# Emotique

## Introduction
Emotique is a futuristic website project based on the idea that robots and AI may become increasingly close to humans and develop curiosity about human emotions in the future.

Based on this concept, the website was designed around the unique idea of “selling emotions.”

This platform provides a space where both AI and robots can explore emotions and purchase them when needed, creating new emotional experiences through shared emotional values.

---

## Technologies Used

- HTML
- CSS
- JavaScript

### Development Details
- HTML was used as the primary markup language to define the website structure.
- JavaScript was implemented to create dynamic and interactive features.
- CSS was used to enhance the design, styling, and overall user experience.

---

The header contains:

  - Home

## Home

## Reviews
